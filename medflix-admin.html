Sauvegarde des données

Exporte et restaure les données de la plateforme. Fais un backup régulièrement pour ne rien perdre.

⬇️ Exporter (Backup)

Télécharge un fichier JSON contenant tous les étudiants, cours, QCM, abonnements et paramètres. Garde ce fichier en sécurité.

Fichier : medflix-backup-[date].json

⬆️ Restaurer

Importe un fichier backup précédent pour restaurer toutes les données. Les données actuelles seront remplacées.

⚠️ Attention
Cette action remplacera toutes les données existantes. Fais d'abord un export si tu veux garder les données actuelles.
